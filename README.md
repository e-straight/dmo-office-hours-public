# DMO Office Hours Handouts

This repository hosts lightweight HTML handouts for GitHub Copilot App office hours.

## Pages

- [Quick start handout](quick-start-handout.html) — first-session guidance, setup tips, and starter prompts.
- [Power user handout](power-user-handout.html) — advanced workflows for agent sessions, orchestration, canvases, automations, model routing, skills, and PR follow-through.

The files are plain static HTML so they can be served directly with GitHub Pages from the repository root.
